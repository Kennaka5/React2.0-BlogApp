Below is a timeline of how to build this apps boiler plate mock up

1. in terminal run 'npx expo-cli init "blog name"'
2. go through the prompts, leave the template blank and name the project
3. install react navigation 'npm install react navigation'


installed json server:

1. cd outside of project: cd ..
2. create new folder for the jsonserver: mkdir json server
3. change into that folder: cd jsonserver
4. create init : npm init
5. say yes sto all questions
6 :npm install json-server ngrok

spin up project:

blog: npm start
jsonserver: npm run db
ngrok: npm run tunnel
